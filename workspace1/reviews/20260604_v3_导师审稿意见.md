# 导师审稿意见

## 审核结论

major_revision

## 主要问题

- 新颖性无法成立：稿件未给出可核验的相关工作综述与参考文献，无法证明“轻量化”“跨对比度”“跨任务联合”三者的组合是否具有明确区别于现有方法的创新贡献，也无法说明相对CMF-Net及其他MRI超分方法的具体增量。
- 方法可行性与可复现性严重不足：网络结构、模块设计、跨对比度交互机制、跨任务定义、损失函数、超分倍率、退化模型、训练策略、推理协议等核心信息缺失，当前方法章节实际上不能支撑同行复现或技术审查。
- 证据链不完整：实验只报告单组主结果和部分效率相对下降，缺少统一协议下的绝对基线结果、数据划分、样本量、评价设置、统计稳健性、消融实验、可视化和失败案例，无法支撑有效性、优势来源和泛化性结论。
- 学术规范性不足：参考文献为空，相关工作章节为空壳；摘要、标题与正文虽较克制，但仍以“联合框架”形式呈现一项尚未被完整定义的方法，存在论文形态已成而学术支撑未成的问题。
- 当前稿件更像“证据边界说明书”而非可投稿论文：其价值在于诚实界定已知与未知，但按论文评审标准，尚未形成完整的研究叙事、方法论闭环和实验闭环。

## 次要问题

- 标题偏长，且“基于当前材料的审慎修订稿”更像内部工作标记，不适合正式投稿标题。
- 摘要中数值结果明确，但未说明超分倍率、比较协议和测试单位，导致结果解释空间过大。
- 引言对研究问题的动机较泛，缺少临床/成像流程层面的具体应用场景支撑。
- 方法章节目前大量采用‘尚未提供’式表述，适合作为内部占位文本，不适合作为对外论文正文。
- 实验章节未说明PSNR/SSIM是在何种裁边、归一化或重建域下计算，评价口径不明。
- 推理时间78 ms缺少batch大小对应关系、warm-up、重复次数和是否含数据传输的定义。
- 结论部分重复了较多‘证据不足’的提醒，可压缩并改为更聚焦的研究结论与局限性陈述。
- 缺少图表：没有框架图、结果表、消融表、可视化图，论文可读性较弱。

## 必须修改

- 补写完整相关工作与参考文献，至少覆盖MRI超分辨率、多对比度/跨模态重建、轻量化重建网络三条主线，并明确本文相对最相关方法的差异点与预期贡献边界。
- 将创新点写成可检验的条目，而非口号式概念。建议明确回答：1）跨对比度机制是什么；2）跨任务具体是哪一任务；3）轻量化来自哪些结构设计；4）每一点分别解决什么问题。
- 重写方法章节，提供完整网络结构、模块公式/伪代码、输入输出定义、张量流、损失函数、训练流程和推理流程，确保他人可复现。
- 明确实验协议：数据来源、训练/验证/测试划分、超分倍率、低分辨率退化模型、预处理、评价指标计算方式、硬件环境、时延统计协议、随机种子。
- 补充统一协议下的完整基线比较，至少给出CMF-Net及其他代表性方法的绝对PSNR/SSIM/参数量/FLOPs/时延，而非仅给相对下降比例。
- 补充消融实验，逐项验证轻量化设计、跨对比度模块、跨任务联合策略、损失项设置对性能与效率的贡献。
- 补充统计稳健性分析，包括多随机种子均值±标准差，必要时进行显著性检验，避免单次结果支撑强结论。
- 补充分场景结果，如不同对比度、不同倍率、不同解剖区域或不同数据子集表现，以验证方法稳定性。
- 补充定性可视化、误差图和失败案例，说明模型在哪些结构细节上有效、在哪些情况下失效。
- 若暂时无法补齐方法与实验证据，应主动降格稿件定位，例如改写为‘阶段性技术报告/证据整理稿’，而不要以正式研究论文口径陈述。
- 正式投稿版需删除内部过程化表述，如‘当前工作区未发现’‘本草稿不编造’等元叙事语言，改为标准学术写法。
- 在摘要、引言与结论中收紧所有超出证据范围的措辞；在未完成统一基线和统计验证前，不宜暗示优于现有方法，只能表述为‘在当前设置下观察到性能-效率折中优势’。
