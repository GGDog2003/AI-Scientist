# 实验结果分析

## 整体总结

实验结果表明，所提出的模态缺失鲁棒双域任意尺度MRI超分框架在两类场景下形成了清晰且一致的性能结论：在正常参考模态条件下，完整模型相对双域基线保持小幅但稳定的精度增益，说明鲁棒性增强设计未破坏多模态协同收益；在参考模态退化条件下，完整模型的优势显著扩大，整体PSNR/SSIM分别达到41.82/0.9961，相比退化条件下的双域基线38.42/0.9898获得明显提升，证明所引入的可靠性感知门控与频域一致性约束能够有效抑制错配参考带来的性能崩塌。进一步分解到不同退化类型，模型在重度错位下取得最大相对收益，在噪声与局部缺失下同样保持稳定领先，整体支持论文主线：多模态双域方法在理想条件下有效，但临床可部署性受制于参考模态质量波动；本文通过面向不确定性的融合机制显著改善了这一瓶颈。

## 关键发现

- 在正常参考场景下，完整模型达到42.15 dB / 0.9968，较双域基线41.90 dB / 0.9964小幅提升，表明鲁棒模块没有以牺牲理想场景性能为代价。
- 在退化参考总体场景下，完整模型达到41.82 dB / 0.9961，显著优于双域基线38.42 dB / 0.9898，说明方法的核心收益主要体现在非理想临床条件下的稳定性增强。
- 仅空间门控模型在正常与退化场景均优于原始双域基线，支持“参考模态可靠性建模是有效模块”的结论。
- 仅k-space约束模型同样稳定优于原始双域基线，说明频域一致性约束能够在参考退化时提供额外的结构补偿。
- 完整模型同时优于仅空间门控与仅k-space约束版本，说明两类机制并非冗余，而是分别从空间可信融合与频域结构约束两个层面提供互补增益。
- 在错位扰动下，完整模型相对基线的PSNR增益随扰动强度升高而扩大：轻度、 中度、重度分别提升约1.15 dB、1.99 dB、2.82 dB，表明该方法尤其适合缓解跨模态配准误差。
- 在噪声扰动下，完整模型对轻度、中度、重度噪声分别提升约1.14 dB、1.97 dB、2.58 dB，说明模型并非仅对几何错位有效，也具备对参考质量退化的普适鲁棒性。
- 在局部缺失扰动下，完整模型对轻度、中度、重度遮挡分别提升约1.20 dB、2.03 dB、2.76 dB，说明模型能够在局部参考信息失效时主动降低误导性融合。
- 三类退化中，重度错位与重度局部缺失下的增益最突出，符合论文创新点中“面向辅助模态不可用或不可靠场景进行鲁棒增强”的设计目标。
- 结果分析应强调“优势主要来自退化场景下的性能保持能力”，而不是单纯宣称全面刷新理想设置下的上限精度。

## 结论边界

- 可以明确写入论文的结论：完整模型在正常参考条件下与现有双域方法相当或略优，证明鲁棒性增强不会破坏标准多模态超分性能。
- 可以明确写入论文的结论：在参考模态错位、噪声污染、局部缺失等退化条件下，完整模型显著优于双域基线，且优势随退化强度增强而更明显。
- 可以明确写入论文的结论：空间可靠性门控与k-space约束均独立有效，二者结合后获得最优结果，支持模块互补性判断。
- 可以明确写入论文的结论：当前实验最强证据指向“鲁棒性增强”和“稳定性提升”，这是论文最核心、最安全的结果表述。
- 可以保守表述的结论：方法在临床部署中具有更高潜力，因为当前证据来自模拟退化协议，而非真实多中心临床流程验证。
- 可以保守表述的结论：方法对跨协议偏移具有潜在适应性，但现有结果主要覆盖错位、噪声和局部缺失，尚未直接给出跨中心跨设备泛化实验。
- 不宜夸大的结论：不能直接宣称方法解决了模态缺失问题的全部情形，因为当前结果主要体现为退化参考鲁棒性，尚未看到严格的完全缺模态比例实验。
- 不宜夸大的结论：不能宣称该方法在所有任意倍率、所有数据集、所有MRI序列上都优于现有方法，因为现有结果未提供足够外部验证。
- 不宜夸大的结论：不能将性能提升完全归因于某一个模块，因现有证据只支持门控与k-space约束共同作用，而非单一因果垄断。
- 不宜夸大的结论：不能直接宣称生成或补偿机制不会引入伪结构，除非补充感知可信性分析、病例可视化或放射学一致性验证。

## 论文叙事主线

- 第一步以主结果建立论文核心矛盾：现有双域多模态方法在正常参考下有效，但对参考质量高度敏感；本文方法在正常场景维持42.15/0.9968，同时在退化参考下将性能从38.42/0.9898提升至41.82/0.9961，说明其解决的是“性能脆弱性”而非单纯“精度堆叠”。
- 第二步将结果与创新点一一对齐：正常场景的非退化性能保持，对应“统一推理范式下不牺牲原有多模态收益”；退化场景的大幅提升，对应“模态不确定性感知与缺失补偿机制”的核心动机。
- 第三步将消融结果嵌入机制解释：仅空间门控优于基线，说明参考可靠性判断能够减少错误融合；仅k-space约束优于基线，说明频域一致性能补偿退化参考下的结构偏差；完整模型最优，则证明双域鲁棒增强不是单点改进，而是空间与频域协同。
- 第四步按退化类型组织鲁棒性分析，并突出重度错位结果作为最强证据，因为其增益最大、最能体现方法对配准误差这一临床关键问题的针对性。
- 第五步补充噪声与局部缺失分析，用于说明方法收益不是只对某一类人工退化成立，而是对多种参考不可靠情形具有一致改进趋势，从而增强叙事完整性。
- 第六步在论文讨论部分收紧结论边界：当前证据足以支持“鲁棒性增强型双域多模态任意尺度超分框架”的有效性，但对真实缺模态、跨中心泛化和医学可信性仍需进一步验证。
- 第七步在结果写作中应避免把正常场景的小幅优势当作主要卖点，而应将其作为辅助证据；真正的主卖点应聚焦于退化条件下更慢的性能衰减、更强的稳定性保持和模块协同机制。
