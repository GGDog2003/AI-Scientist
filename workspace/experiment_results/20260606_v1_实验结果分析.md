# 实验结果分析

## 整体总结

当前结果已经足以支撑一条清晰且成立的论文主线：R2K-SR在正常参考场景下维持与双域方法相当或略优的性能，而在参考模态出现错位、噪声或局部缺失时，凭借可靠性感知的空间-k空间协同机制显著提升鲁棒性，其中对错位尤其有效。论文中最应该强调的不是正常场景下约0.25 dB的小幅增益，而是退化场景下3.40 dB的总体提升、以及从轻度到重度退化都更平缓的性能下降趋势。就写作边界而言，可以较强地宣称“鲁棒性提升”和“模块互补性成立”，但应避免把当前结果扩展为“全面超越SOTA”“可靠性估计已被直接验证”或“临床泛化已被证明”。

## 关键发现

- 正常参考场景下，完整模型达到42.15 dB / 0.9968，优于baseline_dual_domain的41.90 dB / 0.9964，说明引入可靠性建模并未牺牲性能上限。
- 退化参考场景下，完整模型达到41.82 dB / 0.9961，显著优于baseline_dual_domain的38.42 dB / 0.9898，是整篇论文最关键的结果支撑。
- 相较仅空间门控和仅k-space约束，完整模型在退化场景分别再提升2.55 dB和2.21 dB，说明两类模块不是替代关系，而是协同关系。
- 在三类退化中，错位场景的收益最大：轻/中/重错位下完整模型较基线分别提升1.15/1.99/2.82 dB，且退化越重，优势越明显。
- 噪声场景下方法同样稳定：轻/中/重噪声下分别提升1.14/1.97/2.58 dB，说明可靠性感知不仅针对几何错位，也能抑制质量污染参考。
- 局部缺失场景下完整模型较基线分别提升1.20/2.03/2.76 dB，表明门控机制能够在参考信息局部失效时抑制错误迁移。
- 三类退化都呈现共同趋势：退化越重，基线下降越快，而完整模型下降更缓。这一“降幅差异”是鲁棒性叙事的核心证据。
- 单模块消融表明，空间可靠性门控本身有效，k-space约束本身也有效，但只有二者联合时才能接近正常参考场景表现。
- 从正常到退化场景，完整模型仅下降0.33 dB PSNR、0.0007 SSIM；而基线下降3.48 dB、0.0066 SSIM，说明R2K-SR显著缩小了参考退化导致的性能损失。
- 结果与创新动机高度对齐：论文不是证明“再加一个注意力模块更强”，而是证明“显式建模参考可靠性”能够解决多模态MRI超分在不可信参考下的失稳问题。

## 结论边界

- 可以写：R2K-SR在正常参考场景下达到不低于双域基线的重建性能，并有小幅优势。依据现有结果，完整模型相对baseline_dual_domain在正常场景提升约0.25 dB PSNR、0.0004 SSIM，结论应表述为“保持竞争力并略有提升”，不宜夸大为“显著超过现有方法”。
- 可以写：R2K-SR在参考模态退化场景下表现出更强鲁棒性，这是当前结果中最核心、最有力的论文主结论。完整模型在退化场景相对双域基线提升3.40 dB PSNR、0.0063 SSIM，且三类退化下均成立。
- 可以写：错位退化是最能体现方法优势的场景。重度错位下完整模型相对基线提升2.82 dB PSNR、0.0056 SSIM，说明可靠性感知机制对配准误差具有较强抑制作用。
- 可以写：空间可靠性门控与k-space约束具有互补性。因为仅空间门控、仅k-space约束、完整模型三者结果呈逐级提升，且完整模型在正常与退化场景均最好。
- 可以写：频域一致性约束在不可靠参考条件下仍然有效，但必须与可靠性建模协同才能释放最大收益。该结论有单模块和完整模型对比支撑。
- 可以保守写：方法对噪声污染和局部缺失同样有效，且稳健性提升具有一致性。因为三类退化、轻中重三级结果均单调优于基线。
- 只能保守表述：方法具有“临床场景友好性”或“实际应用潜力”，因为当前仅有重建指标与受控退化协议，尚缺真实临床采集失配、跨中心数据和下游任务验证。
- 只能保守表述：可靠性图真正学习到了参考可信度分布。当前没有AUC、ECE、Brier、可视化对齐或伪标签评估，因此不能直接宣称“可靠性估计准确”。
- 不能直接写：方法在所有SOTA方法上都显著领先。现有结果只覆盖双域基线和内部消融，不包含HFST、DMASR等外部方法正式对比，也没有统计显著性检验。
- 不能直接写：方法对任意尺度超分的优势已被充分证明。当前给出的结果未按2x/3x/4x/连续尺度拆分，叙事上只能沿用方法设定，不能把“任意尺度优势”写成实验定论。
- 不能直接写：方法在边缘保真度、频域一致性、结构保持性上全面优于现有方法。实验设计方案虽包含HFEN、Edge-SSIM、k-space误差和下游分割指标，但当前结果尚未提供这些数值。
- 不能直接写：完整模型对所有退化强度都实现“显著”提升。若论文使用“显著”措辞，需补充病例级统计检验；在现阶段更稳妥的写法是“一致优于”或“明显优于”。

## 论文叙事主线

- 第一段问题定义：现有多模态MRI超分通常默认参考模态高质量且已良好配准，但真实场景中常伴随错位、噪声和局部缺失，导致参考信息可能由帮助项转为干扰项。
- 第二段文献承接：DMASR证明空间域与k-space双域建模有效，HFST证明结构先验和跨模态信息对高频恢复重要；但两条路线都缺少对“不可靠参考”条件的系统建模与验证。
- 第三段方法主张：R2K-SR通过参考模态可靠性感知的空间门控与k-space一致性协同机制，在参考可信时充分利用跨模态信息，在参考失真时主动抑制错误引导。
- 第四段主结果先放总体表：正常参考场景下性能不降反升，退化参考场景下优势显著扩大。这里要把42.15/0.9968与41.82/0.9961放在最前面，形成“上限不丢、鲁棒性更强”的第一印象。
- 第五段展开鲁棒性主实验：按错位、噪声、局部缺失三类退化分别报告轻中重结果，重点强调完整模型的性能下降曲线更平缓，且重度错位场景收益最大。
- 第六段做机制归因：用单模块消融说明，空间可靠性门控负责抑制不可信参考传播，k-space约束负责维持频域一致性，二者联合带来最优结果。
- 第七段把结果与创新点逐项对齐：创新点1是可靠性感知建模，对应退化参考下显著更稳；创新点2是目标函数中的频域约束，对应单独k-space模块已有效、联合后进一步提升。
- 第八段图表组织建议：Table 1放正常/退化总体对比；Table 2放三类退化轻中重拆分；Table 3放消融；Figure 1放性能-退化强度曲线；Figure 2放典型错位样例、误差图和可靠性热图。
- 第九段讨论边界：强调结果来自受控退化协议，证明的是“对合成但具有现实指向性的参考失真更稳健”；对真实跨中心、真实失配与临床终点的外推需保持克制。
- 第十段结论收束：本文的贡献不在于追求正常场景下微小数值增益，而在于补足多模态MRI超分长期缺失的‘参考可靠性鲁棒性’证据链，从而使方法叙事更完整。
