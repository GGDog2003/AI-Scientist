# 实验结果分析

## 整体总结

实验结果表明，所提出的模态缺失鲁棒双域任意尺度MRI超分框架在两类场景下同时成立：其一，在正常参考模态条件下，模型未因鲁棒性设计牺牲基础重建性能，反而相对双域基线取得小幅但一致的增益；其二，在参考模态退化条件下，模型相对基线获得显著更大的PSNR/SSIM提升，且优势会随退化强度增加而进一步扩大。整体上，结果支持论文主叙事：传统双域多模态方法在理想配准下有效，但对辅助模态质量高度敏感；通过引入参考可靠性感知门控与频域一致性约束，可将多模态互补性从“高性能但脆弱”推进到“高性能且更稳健”的临床可部署范式。

## 关键发现

- 在正常参考场景下，完整模型达到42.15 dB / 0.9968，优于双域基线的41.90 dB / 0.9964，说明鲁棒性增强模块并未破坏标准多模态超分设定下的重建上限，反而带来小幅稳定增益。
- 在退化参考场景下，完整模型达到41.82 dB / 0.9961，显著优于双域基线的38.42 dB / 0.9898，PSNR提升3.40 dB、SSIM提升0.0063，表明所提方法的主要收益集中体现为对异常辅助模态输入的稳定性增强。
- 仅空间门控版本在正常/退化场景下分别达到41.97 dB / 0.9965和39.27 dB / 0.9921，证明参考模态可靠性建模本身具有独立有效性，能够抑制低质量辅助信息对主模态重建链路的负迁移。
- 仅k-space约束版本在正常/退化场景下分别达到42.01 dB / 0.9966和39.61 dB / 0.9928，说明频域一致性约束能够在参考模态退化时维持高频结构恢复稳定性，是双域鲁棒增强的重要组成部分。
- 完整模型同时优于两个单模块版本：在退化参考下，相比仅空间门控再提升2.55 dB，相比仅k-space约束再提升2.21 dB，说明空间可靠性筛选与频域结构约束不是冗余设计，而是具有明确互补性。
- 错位扰动是最能暴露原始多模态双域方法脆弱性的因素，也是完整模型优势最大的场景。在重度错位下，基线为37.44 dB / 0.9882，完整模型为40.26 dB / 0.9938，PSNR提升2.82 dB，支持“跨模态偏移会破坏融合有效性，而可靠性感知机制可显著缓解该问题”的核心创新论点。
- 噪声退化下，完整模型在轻/中/重三档分别较基线提升1.14 dB、1.97 dB和2.58 dB；局部缺失下分别提升1.20 dB、2.03 dB和2.76 dB，说明方法并非只对一种特定退化有效，而是在多类参考异常上呈现一致鲁棒性趋势。
- 随着退化程度由轻到重增加，完整模型相对基线的性能差距持续扩大，表明所提方法并非仅在边缘条件下带来偶然收益，而是在难例场景中更具稳定价值，这一点与“面向临床复杂采集条件的鲁棒增强”定位高度一致。
- 正常场景下的提升幅度明显小于退化场景下的提升幅度，意味着论文结果分析应强调方法的首要贡献是鲁棒性提升而非单纯追求SOTA式的理想条件峰值性能刷新。
- 从现有结果可归纳出明确叙事：基线方法在辅助模态质量可靠时已经较强，因此正常场景增益有限；而一旦参考模态失真，未加约束的多模态融合会引入错误补偿，完整模型则通过可靠性控制和频域约束有效抑制该问题。

## 结论边界

- 可以明确写入论文的结论：完整模型在正常参考条件下保持与双域基线相当或更优的性能，说明鲁棒设计没有以牺牲基础精度为代价。
- 可以明确写入论文的结论：在错位、噪声、局部缺失三类参考退化下，完整模型均显著优于基线，且在中重度退化下优势更大，说明方法具有跨退化类型的一致鲁棒性。
- 可以明确写入论文的结论：空间可靠性门控与k-space一致性约束均能独立带来提升，完整模型进一步优于任一单模块版本，因此两者具有互补贡献。
- 可以较强表述但应保持学术克制的结论：错位是对多模态双域融合破坏最强的退化因素之一，所提方法在该场景收益最大，因而特别适合处理弱配准或轻度跨模态偏移条件下的临床应用。
- 只能保守表述的结论：方法提升了临床可部署性。当前结果支持“更接近临床部署需求”或“缓解临床落地中的模态质量敏感性”，但不能直接上升为“已满足临床部署”或“具备临床可用性”，因为尚未看到跨中心、跨设备、真实缺失协议与医生评估证据。
- 只能保守表述的结论：方法对模态缺失具有鲁棒性。现有数据直接覆盖的是错位、噪声和局部缺失，若论文标题或摘要使用“模态缺失鲁棒”，正文需要注明当前实验主要验证的是参考退化与局部信息缺失场景；若未给出完整模态缺失比例实验，不宜夸大为“完全模态缺失条件下显著有效”。
- 不能直接写成强结论的内容：所提门控一定学到了真实的医学可信度分布。当前只有性能结果，没有门控权重可视化、统计分布或与退化强度的相关性分析，因此只能说门控机制在功能上有效，而不能声称其解释性已被充分验证。
- 不能直接写成强结论的内容：k-space约束恢复了更真实的组织结构。现有证据主要是PSNR/SSIM提升，除非补充边缘保真、频谱能量分析、局部结构误差图或医生主观评分，否则应表述为“提升频域一致性并改善重建质量”，避免直接宣称“医学结构真实性显著提升”。
- 不能夸大的内容：方法已经全面优于所有现有任意尺度MRI超分方法。当前对比对象以双域基线及内部变体为主，若缺少更广泛SOTA比较，则论文中应限定为“相对于强双域多模态基线具有更强鲁棒性”。
- 不能夸大的内容：方法在所有退化强度和所有数据分布上都稳定有效。当前结果仅能支持给定实验协议下的稳定性趋势，跨数据集泛化和极端协议外退化仍需进一步验证。
- 建议在论文中主动写明的边界：性能增益主要体现于异常辅助模态条件；在理想参考条件下，提升幅度较小，因此方法价值集中在可靠性增强而非重构主干的大幅刷新。
- 建议在讨论部分写明的风险边界：若缺失补偿或门控模块过强，可能引入额外计算开销或错误抑制；当前结果证明整体收益为正，但尚需补充复杂度、推理时间和稳定性统计来完整支撑工程部署结论。

## 论文叙事主线

- 第一步建立问题张力：现有双域多模态任意尺度MRI超分在理想条件下性能强，但其有效性隐含依赖辅助模态可用、准确配准和低噪声质量，这一前提与真实临床采集条件存在显著落差。
- 第二步用正常参考结果说明本文并非通过牺牲上限换取鲁棒性：完整模型在正常场景下从41.90 dB提升至42.15 dB，证明所提机制与原始双域互补是兼容的，具备作为通用推理框架的合理性。
- 第三步用退化参考总表给出主结论：当辅助模态异常时，基线性能明显下滑，而完整模型仍保持41.82 dB / 0.9961，相比基线取得3.40 dB / 0.0063的大幅优势。这一段应作为结果部分的中心段落，直接支撑论文创新点。
- 第四步将主结论拆解到模块层面：仅空间门控有提升，说明“是否使用辅助信息”需要由质量驱动；仅k-space约束也有提升，说明“即使使用辅助信息，也需要频域一致性防止错误高频传播”；完整模型最优，则自然导出“双机制协同”这一设计逻辑闭环。
- 第五步将鲁棒性结论细分到退化类型，形成与动机完全对齐的分析结构：错位对应弱配准问题，噪声对应低质量辅助扫描，局部缺失对应不完整参考信息。三类场景均有效，说明方法并非为单一失真定制。
- 第六步重点突出错位场景，因为这最贴合文献动机中的跨模态偏移与配准误差问题。可将重度错位结果单独作图或单列子节，强调基线由37.44 dB下降而完整模型仍达40.26 dB，说明可靠性感知融合显著缓解了错误对齐导致的负迁移。
- 第七步强调“退化越强，收益越大”的趋势，这是论文叙事中非常关键的证据。该趋势证明所提模块并非在简单样本上重复已有能力，而是在困难条件下发挥作用，因而与临床可部署性目标直接相关。
- 第八步在消融分析中按“功能归因”组织，而非简单罗列表格：空间门控负责抑制不可靠辅助特征，k-space约束负责稳定频域细节恢复，二者叠加后同时提升鲁棒性与最终重建质量。这种写法能让创新点、模块设计和结果现象逐项对齐。
- 第九步在可视化叙事中建议对应三种图：总体性能柱状图用于展示正常/退化双场景；退化强度折线图用于展示收益随扰动增强而扩大；典型病例误差图用于展示基线在错位或局部缺失下出现的伪细节/结构偏差，以及完整模型的抑制效果。
- 第十步在论文结论段采用双层表达：强结论写“所提方法显著提升参考退化条件下的鲁棒性，且不损害正常条件性能”；保守结论写“结果表明该框架朝临床复杂条件下的可部署MRI超分迈出关键一步”，避免越过当前证据边界。
- 第十一步在讨论部分主动承接局限性，以增强论文可信度：当前验证主要集中于合成退化协议，尚需补充真实跨中心偏移、完整模态缺失、效率开销与可信性评估。这种写法不会削弱主结论，反而能凸显本文贡献边界清晰、问题定义准确。
- 第十二步最终将整篇论文主线收束为一句可执行叙事：多模态双域任意尺度MRI超分本身有效，但对辅助模态质量脆弱；本文通过可靠性感知与频域一致性协同设计，使模型在理想场景不退化、在异常场景显著更稳健，从而补上现有方法面向真实临床应用的关键短板。
