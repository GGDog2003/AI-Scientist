# 实验设计方案

## 实验目标

验证配准不确定性感知的多模态脑MRI任意尺度超分方法 RUA-SR 是否能够在标准配准数据、人工错配数据以及跨中心分布偏移条件下，相比现有多模态任意尺度超分基线更稳定地提升目标模态重建质量，并显著降低由跨模态轻微失配引起的伪结构迁移。实验重点回答四个研究问题：RQ1，RUA-SR 在标准测试集上是否至少维持或小幅提升 PSNR/SSIM/结构保真指标；RQ2，在可控平移、旋转、局部弹性形变错配下，RUA-SR 是否具有更低的性能退化斜率和更小的边缘错误；RQ3，不确定性图是否学习到与真实错配位置相关的可靠性分布，而非退化为普通注意力；RQ4，局部对齐估计器、不确定性预测头和不确定性引导动态融合三者是否形成互补贡献，并在不同倍率与不同模态组合下保持稳定收益。

## 数据集

- IXI brain MRI dataset：使用 T1、T2、PD 三模态；按 subject 划分 train/val/test，建议 70%/10%/20%；用于主体训练与标准测试。
- BraTS 2021 or BraTS 2020：使用 T1、T1ce、T2、FLAIR，选择其中两模态或三模态作为输入组合，目标模态固定为 T1 或 T2；用于跨数据集泛化与病灶区域鲁棒性验证。
- fastMRI-Brain multicontrast subset 或自建临床多序列脑MRI数据：若可获得多序列配准较弱数据，用于验证真实失配场景；若不可获得，则作为可选扩展实验集。
- 基于 IXI 与 BraTS 构建的 Synthetic Misregistration Benchmark：在测试阶段对辅助模态施加分级错配，包括平移 {1,2,3,4,5} 像素、旋转 {1,3,5,7,10} 度、缩放 {0.98,1.02}、局部弹性形变强度 {low, mid, high}，并保留目标模态不变，用于定量鲁棒性评测。
- 任意尺度超分设置下的降采样协议数据：对 HR 轴向切片或 3D 体数据使用 k-space 截断或图像域 bicubic/BSpline 降采样生成 LR；训练倍率从连续区间 x1.1-x4.0 随机采样，测试倍率覆盖 x2.0、x2.5、x3.0、x4.0。

## 对比基线

- Bicubic interpolation：最基础重建下界。
- EDSR 或 RCAN 单模态版本：验证多模态信息增益。
- SwinIR 或 HAT 单模态 MRI 适配版本：作为强单模态 Transformer 基线。
- High-frequency Structure Transformer for MRI Super-resolution：作为结构增强型先进基线。
- Dual-domain Multi-modality Brain MRI Arbitrary-scale Super-resolution Network：核心主比较对象，记为 DDMM-ASR。
- DDMM-ASR + naive concatenation fusion：去除其原有复杂交互，仅保留简单多模态输入，用于说明融合策略重要性。
- DDMM-ASR + deformation alignment only：仅加入局部对齐模块，不加入不确定性建模。
- DDMM-ASR + uncertainty gating only：仅加入不确定性引导融合，不加入显式局部对齐。
- DDMM-ASR + confidence-blind deformable fusion：使用可变形融合但不显式预测不确定性，用于区分不确定性机制与普通可变形注意力。
- RUA-SR full model：局部对齐估计器 + 不确定性预测头 + 不确定性引导双域融合 + 模拟错配一致性正则。

## 评价指标

- PSNR：全图像重建质量主指标。
- SSIM：结构相似性主指标。
- NRMSE：归一化误差。
- HFEN：高频误差范数，衡量边缘和细节恢复偏差。
- GMSD 或 Gradient-Magnitude Similarity：衡量梯度结构保真。
- LPIPS 或 DISTS（若采用 2D 切片评估）：补充感知一致性，但仅作次要指标。
- ROI-PSNR/ROI-SSIM：在灰白质边界、脑室边界、病灶区域上单独评估，验证伪结构抑制。
- Edge Error Map Statistics：基于 Sobel/Canny 边缘后计算边缘 MAE、边缘 F1、边缘误检率，用于体现错误纹理迁移。
- Misregistration Sensitivity Curve：以错配强度为横轴，PSNR/SSIM/HFEN 为纵轴，比较性能退化斜率。
- Uncertainty-Misalignment Correlation：预测不确定性图与已知错配区域掩码的 Spearman/Pearson 相关系数、AUC，用于验证不确定性语义有效性。
- Calibration Metrics for uncertainty：ECE、Brier score 或 NLL，将像素级重建残差分桶后评估不确定性校准性。
- Runtime and Params：参数量、FLOPs、单张切片推理时间，证明鲁棒性收益与额外开销的平衡。
- 统计检验：对主要指标进行配对 t 检验或 Wilcoxon signed-rank test；对多错配强度条件采用 repeated-measures ANOVA 或 Friedman test，并报告 p 值与 Cohen's d/Cliff's delta 效应量。

## 消融实验

- A1：完整模型 RUA-SR vs 去除不确定性预测头，验证不确定性分支独立贡献。
- A2：完整模型 vs 去除局部对齐估计器，验证局部偏移建模作用。
- A3：完整模型 vs 去除动态抑制/选择门控，仅直接融合，验证不确定性引导融合必要性。
- A4：完整模型 vs 仅对齐无不确定性 vs 仅不确定性无对齐，分析两模块互补性。
- A5：不确定性图作用方式比较：乘性抑制、加性偏置、softmax 门控、top-k token/filter 选择。
- A6：不确定性监督方式比较：仅隐式重建监督、残差伪监督、错配区域二值伪标签监督、联合校准损失。
- A7：模拟错配一致性正则强度 lambda 比较，取 {0, 0.01, 0.05, 0.1, 0.5}。
- A8：训练阶段错配增强强度比较：无增强、轻度、中度、课程式增强，验证鲁棒性形成机制。
- A9：局部对齐模块复杂度比较：3x3 offset conv、deformable conv、局部相关性搜索，权衡性能与成本。
- A10：双域分支消融：仅空间域、仅频域/结构域、双域联合，确认原始双域框架与新模块的协同关系。
- A11：不同输入模态组合：T1->T1 单模态、T1+T2->T1、T1+PD->T1、T1+T2+PD->T1；BraTS 上测试 T1+FLAIR->T1ce 等组合。
- A12：不同放大倍率：x2.0、x2.5、x3.0、x4.0 以及训练外倍率如 x3.5，验证 arbitrary-scale 稳定性。
- A13：不同错配类型：仅平移、仅旋转、仅弹性形变、混合错配，验证鲁棒性来源。
- A14：不同测试域：标准 IXI、人工错配 IXI、跨数据集 BraTS，验证域外稳定性。
- A15：不确定性图有效性分析：高不确定性区域屏蔽后重建误差是否显著高于低不确定性区域；绘制 reliability diagram。
- A16：缺失模态场景：随机丢失一个辅助模态并以 mask token/零填充处理，测试不确定性机制是否降低模态缺失敏感性。

## 代码模块

- configs/experiment_config.py：统一保存数据路径、倍率范围、模态组合、错配增强参数、损失权重、随机种子。
- data/dataset_ixi.py：IXI 数据读取、subject 划分、切片采样、归一化、模态配对。
- data/dataset_brats.py：BraTS 数据读取、脑区裁剪、病灶 ROI 提取、模态组合接口。
- data/degradation.py：任意尺度降采样、bicubic/BSpline/k-space 降质生成。
- data/misregistration.py：平移、旋转、缩放、局部弹性形变、混合错配生成器，并输出真值位移场或错配掩码。
- models/encoders.py：各模态共享或非共享编码器。
- models/dual_domain_backbone.py：复现 DDMM-ASR 主干的空间域与频域/结构域双分支。
- models/local_alignment.py：轻量局部对齐估计器，输出 offset/局部形变参数。
- models/uncertainty_head.py：像素级或特征级不确定性图预测头。
- models/dynamic_fusion.py：不确定性引导的跨模态动态抑制、加权选择与门控融合模块。
- models/arbitrary_upsampler.py：连续坐标 MLP 或 implicit upsampling 模块，实现 arbitrary-scale 输出。
- models/rua_sr.py：完整 RUA-SR 模型封装。
- losses/reconstruction_loss.py：L1、Charbonnier、SSIM loss、多尺度重建损失。
- losses/uncertainty_loss.py：heteroscedastic loss、校准损失、残差一致性约束。
- losses/structure_consistency_loss.py：梯度损失、边缘一致性、频域结构一致性损失。
- losses/misreg_consistency_loss.py：模拟错配一致性正则，约束对轻微错配扰动的稳定输出。
- trainers/train_baseline.py：训练单模态与基线模型。
- trainers/train_rua_sr.py：训练完整模型与消融版本，支持 AMP、DDP、早停、checkpoint。
- evaluation/metrics_image.py：PSNR、SSIM、NRMSE、HFEN、GMSD、LPIPS 等实现。
- evaluation/metrics_uncertainty.py：ECE、Brier、uncertainty-error correlation、risk-coverage curve。
- evaluation/robustness_eval.py：按错配强度批量评估并输出敏感性曲线数据。
- evaluation/stat_tests.py：配对 t 检验、Wilcoxon、ANOVA/Friedman、效应量计算。
- visualization/plot_error_maps.py：重建误差图、边缘误差图、伪结构热图可视化。
- visualization/plot_uncertainty_maps.py：显示不确定性图与错配区域叠加图。
- visualization/plot_sensitivity_curves.py：绘制错配强度-性能曲线、risk-coverage curve、校准图。
- scripts/preprocess_ixi.py：原始数据预处理、重采样、配准检查、数据列表生成。
- scripts/preprocess_brats.py：BraTS 预处理、ROI 生成、划分文件导出。
- scripts/run_train.sh：一键训练基线与主模型。
- scripts/run_eval.sh：一键生成标准测试、鲁棒性测试、消融结果与统计检验。
- utils/reproducibility.py：固定随机种子、记录 git hash、保存环境依赖。
- utils/logger.py：TensorBoard/WandB 日志与表格结果导出。
- utils/checkpoint.py：模型保存与最优模型筛选。

## 执行步骤

- 步骤1：准备 IXI 与 BraTS 数据，统一重采样到固定空间分辨率，执行 skull stripping 或采用官方预处理版本，按 subject 级别划分训练、验证、测试集，确保同一受试者不跨集合。
- 步骤2：确定目标任务配置，例如以 T1 为重建目标模态，辅助模态使用 T2 和 PD；BraTS 上可设定 T1ce 为目标，T1/T2/FLAIR 为辅助输入。为每组实验保存 json/yaml 配置文件。
- 步骤3：使用 preprocess_ixi.py 与 preprocess_brats.py 生成标准化数据索引、切片缓存和 ROI 掩码；对每个切片做 z-score 或 min-max 归一化，并记录模态名称与 subject id。
- 步骤4：通过 degradation.py 构建 LR-HR 训练对。训练时随机采样连续倍率 x1.1-x4.0；测试固定倍率 x2.0、x2.5、x3.0、x4.0。若强调 MRI 真实性，优先采用 k-space 截断降采样；同时保留 bicubic 降采样设置作为补充实验。
- 步骤5：通过 misregistration.py 为辅助模态生成训练期模拟错配增强。建议 50% 概率施加轻度错配，30% 中度，20% 重度；目标模态保持不变。输出已知错配参数与近似区域掩码供不确定性评估使用。
- 步骤6：复现所有基线。首先训练 Bicubic、EDSR/RCAN、SwinIR/HFST、DDMM-ASR，并在验证集上调到论文级最优设置。每个方法至少运行 3 个随机种子，推荐 5 个随机种子以支持显著性检验。
- 步骤7：实现并训练 RUA-SR 完整模型。初始损失建议为 L = L_rec + 0.1 L_struct + 0.05 L_unc + 0.05 L_misreg；优化器使用 AdamW，初始学习率 2e-4，batch size 根据显存设为 8-16，训练 300 epochs，余弦退火调度。
- 步骤8：按消融计划逐个关闭或替换模块，训练 A1-A16 所列模型。为保证公平性，除被消融组件外其余超参数、数据划分和训练策略保持一致。
- 步骤9：在标准测试集上评估 PSNR、SSIM、NRMSE、HFEN、GMSD、ROI 指标，并生成重建可视化。为每个模型输出平均值、标准差、95% 置信区间。
- 步骤10：构建 Synthetic Misregistration Benchmark，在测试阶段系统性施加分级平移、旋转、缩放和弹性形变，仅扰动辅助模态。记录每个强度下的全部指标，绘制 misregistration sensitivity curves，并计算性能退化斜率。
- 步骤11：评估不确定性图的语义有效性。将预测不确定性与已知错配区域、像素重建残差、边缘误差区域进行相关性分析，计算 AUC、Spearman 相关和 ECE，并绘制 reliability diagram 与 risk-coverage curve。
- 步骤12：执行统计检验。对完整模型与主基线 DDMM-ASR 在各测试集上的 PSNR、SSIM、HFEN 使用配对 t 检验；若正态性不满足则改用 Wilcoxon。对多错配强度结果使用 repeated-measures ANOVA 或 Friedman test，并报告 p 值与效应量。
- 步骤13：生成论文级图表。至少包括：模型总图、局部对齐与不确定性融合子模块图、主结果表、消融表、错配鲁棒性曲线图、不确定性可视化图、边缘误差热图、不同倍率与模态组合柱状图、参数量与性能折中图。
- 步骤14：执行失败案例分析。挑选病灶附近、脑室边界、严重错配样本，对比 DDMM-ASR 与 RUA-SR 的重建、误差图与不确定性图，分析何种情况下局部对齐不足或不确定性过度抑制导致细节损失。
- 步骤15：开展跨数据集泛化实验。使用 IXI 训练模型，在 BraTS 或其他中心数据上直接测试；若模态定义不完全一致，则选择最接近模态组合并统一强度标准化，比较域外鲁棒性。
- 步骤16：记录复现实验信息，包括随机种子、GPU 型号、PyTorch 版本、训练时长、最佳 checkpoint 路径、配置文件哈希；通过 utils/reproducibility.py 自动保存到 results 目录。
- 步骤17：将 execution 结果与代码模块建立映射：数据准备对应 data 与 scripts 模块，模型训练对应 models、losses、trainers，标准评测对应 evaluation/metrics_image.py，鲁棒性基准对应 data/misregistration.py 与 evaluation/robustness_eval.py，统计分析对应 evaluation/stat_tests.py，可视化对应 visualization 模块。
- 步骤18：最终交付一个可直接运行的命令序列，例如 preprocess -> train_baselines -> train_rua_sr -> eval_standard -> eval_robustness -> run_stats -> export_figures，确保实验闭环可重现。
