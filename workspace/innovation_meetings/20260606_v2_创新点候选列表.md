# 创新点候选列表

## 组会汇报摘要

当前文献差异表明，已有工作一侧重多模态、双域与任意尺度统一建模，另一侧重高频结构增强与Transformer细节恢复，但二者都未充分解决临床更真实的三个问题：缺失模态、配准误差与高频真实性。基于此，最推荐主线是以Dual-domain Multi-modality Brain MRI Arbitrary-scale Super-resolution Network为主基线，吸收High-frequency Structure Transformer的高频结构建模思想，提出面向缺失模态的双域高频结构任意尺度超分网络。核心改动包括：在双域多模态主干上增加高频结构提取分支，利用梯度/拉普拉斯或频域特征强化边界恢复；设计模态置信门控与配准噪声感知对齐，抑制错误跨模态纹理迁移；训练阶段加入模态dropout与一致性约束，使模型在完整模态和缺失模态场景均稳定工作；目标函数联合像素重建、边缘结构、高频一致性和鲁棒蒸馏损失。方法总图建议按“多模态输入—可靠对齐—双域主干—高频结构增强—置信融合—任意尺度重建”绘制。实验上应以原双域多模态基线、原高频结构Transformer、常规CNN/Transformer超分模型为对照，并重点补充四类消融：高频分支有效性、门控融合有效性、缺失模态训练策略有效性、配准扰动鲁棒性。若数据与时间有限，优先做二维切片级验证和人工缺失/错配实验，即可较快形成完整论文主线。

## 候选创新点

### 面向缺失模态的双域高频结构任意尺度超分网络（MD-HFSR）
- 动机：现有两篇工作分别强调多模态双域任意尺度建模与高频结构Transformer增强，但二者的能力边界互补且缺口明确：前者在多模态利用和倍率泛化上占优，却对缺失模态、配准误差与高频真实性讨论不足；后者在边界与纹理恢复上更强，却缺少任意尺度建模、多模态互补与模态缺失适配。因此可在Dual-domain Multi-modality Brain MRI Arbitrary-scale Super-resolution Network基础上，引入高频结构先验分支、模态置信门控与缺失模态训练机制，形成一条叙事完整且可快速验证的主线：在任意尺度MRI超分中，利用可靠的跨模态结构信息增强高频恢复，同时抑制错配模态带来的伪结构迁移。
- 创新类型：基于现有多模态任意尺度基线的模块增强型创新，具体属于“基线整合+鲁棒性扩展+目标函数改进”三位一体方案。改动对象为Dual-domain Multi-modality Brain MRI Arbitrary-scale Super-resolution Network；新增模块包括高频结构提取分支、模态置信门控融合模块、配准噪声感知跨模态对齐模块；新增目标函数包括高频一致性损失、边缘结构损失、模态鲁棒蒸馏或模态dropout一致性损失。方法总图可画为：输入多模态低分辨率MRI及目标放大尺度 -> 各模态浅层编码 -> 配准噪声感知对齐 -> 共享双域主干 -> 高频结构支路提取梯度/拉普拉斯/频带特征 -> 置信门控跨模态融合 -> 连续隐式上采样 -> 重建头输出。关键消融包括：去掉高频支路、去掉门控、去掉双域、去掉模态dropout训练、不同高频提取器、不同缺失模态比例、不同配准扰动强度。
- 预期收益：预期可同时提升三类指标：第一，定量重建指标，如PSNR、SSIM以及任意非整数倍率下的稳定性；第二，结构保真指标，如梯度误差、边缘保持指数、灰白质边界清晰度；第三，临床可用性指标，如在缺失模态和轻度失配条件下的鲁棒性。叙事上可形成较强论文主线：从“多模态可提升上限但不稳定”过渡到“高频增强可提细节但易受伪结构污染”，最终提出“可靠性约束下的多模态高频结构超分”。
- 风险点：模块较多，若数据规模有限，训练不稳定或收益分散，需控制参数量并采用分阶段训练。；高频支路可能放大MRI噪声和伪影，必须结合置信门控与边缘一致性约束。；若公开数据集模态配准本身较好，配准噪声感知模块的增益可能不显著，需要人为构造错配实验。；缺失模态训练若设计不当，完整模态场景下的性能上限可能下降。；任意尺度连续上采样与Transformer/双域模块组合后，推理成本可能偏高，需要报告FLOPs和推理时间。
- 证据依据：Dual-domain Multi-modality Brain MRI Arbitrary-scale Super-resolution Network证明了多模态+双域+任意尺度框架在脑MRI超分中具备较强性能，但其局限性明确指出缺失模态、配准误差和极端倍率稳定性不足。；High-frequency Structure Transformer for Magnetic Resonance Image Super-resolution证明了显式高频结构建模可改善边界锐利度与纹理恢复，但其局限性指出噪声敏感、缺乏频率-解剖一致性与多模态支撑。；两篇文献的创新方向互补：一篇解决“信息来源广度与倍率泛化”，另一篇解决“结构细节质量”，组合后具有直接理论动机。；两篇文献的限制共同指向‘视觉锐利但医学不真实’与‘多模态误导’风险，因此引入置信门控和结构一致性损失具有文献支撑。

### 配准不确定性感知的多模态脑MRI任意尺度超分（RUA-SR）
- 动机：多模态MRI超分方法的主要工程瓶颈并非仅来自网络表达能力，而是来自模态间轻微失配导致的错误纹理迁移。现有双域多模态任意尺度方法默认跨模态信息可直接融合，但文献已明确指出配准误差会引入伪结构。相比再增加一个复杂主干，优先围绕“配准鲁棒性”做定向改造，更易形成清晰问题定义与实验闭环。该方案适合快速验证，因为只需在Dual-domain Multi-modality Brain MRI Arbitrary-scale Super-resolution Network上增加不确定性估计与动态融合机制，即可建立明显区别于原始基线的贡献。
- 创新类型：针对基线脆弱环节的鲁棒性机制创新。改动对象为Dual-domain Multi-modality Brain MRI Arbitrary-scale Super-resolution Network；新增模块包括跨模态局部对齐估计器、配准不确定性预测头、基于不确定性的动态特征抑制/选择模块；新增目标函数包括不确定性感知重建损失、结构一致性损失、模拟错配一致性正则。方法总图可画为：多模态输入 -> 每模态编码 -> 局部形变/偏移估计 -> 不确定性图预测 -> 双域特征抽取 -> 不确定性引导的跨模态融合 -> 任意尺度上采样 -> 输出高分辨率图像。消融应覆盖：无不确定性图、仅对齐无不确定性、仅门控无对齐、不同错配强度训练、不同模态组合、不同放大倍率。
- 预期收益：预期在存在模态失配、运动扰动或跨中心数据时获得更稳定的PSNR/SSIM，并减少伪边缘与错误组织纹理。即使在标准测试集上平均指标提升有限，该方案也可通过鲁棒性实验建立独立贡献，例如在人工平移、旋转、局部弹性形变设置下维持更高结构相似性。论文叙事可围绕‘多模态超分的真实短板不是是否融合，而是何时该融合、融合多少’展开。
- 风险点：若数据集配准质量过高，标准测试集上的增益可能较小，需要重点设计错配鲁棒性基准。；不确定性估计若缺少显式监督，可能退化为普通注意力图，需要通过一致性实验验证其语义有效性。；局部对齐模块可能增加训练难度，并与任意尺度上采样目标产生耦合。；若评价体系只使用PSNR/SSIM，不易充分体现鲁棒性优势，需增加边缘错误图和失配敏感性曲线。；若引入形变估计过重，可能削弱快速验证优势，应优先采用轻量局部偏移建模。
- 证据依据：Dual-domain Multi-modality Brain MRI Arbitrary-scale Super-resolution Network的局限性直接指出：跨模态配准误差会引入伪结构或错误纹理迁移。；该文也指出跨医院、跨扫描仪域偏移鲁棒性证据不足，说明建立失配鲁棒方案具有现实意义。；High-frequency Structure Transformer for Magnetic Resonance Image Super-resolution强调边界恢复，但也提示视觉锐利可能不等于医学真实，进一步说明错误跨模态细节迁移是关键风险源。；因此，相比单纯追求更强主干，引入融合可靠性建模更贴合前述文献差异与未解决问题。

### 频率-解剖一致性约束的高频结构Transformer超分（FA-ConsisFormer）
- 动机：高频结构Transformer已证明显式高频建模对MRI边缘恢复有效，但文献同时指出其高频分支易将噪声与伪影误当成可增强细节，且缺少频率-解剖一致性联合约束。相较于直接追求多模态扩展，一个更聚焦的方案是在High-frequency Structure Transformer for Magnetic Resonance Image Super-resolution基础上，增加结构真实性约束，使‘锐利’与‘解剖可信’统一。该方案模块增量小，易训练，适合作为快速发表型工作。
- 创新类型：面向单模态高频结构基线的目标函数与轻量模块改造。改动对象为High-frequency Structure Transformer for Magnetic Resonance Image Super-resolution；新增模块包括解剖边缘先验编码器、频段选择门控模块、低频-高频一致性交互块；新增目标函数包括频率一致性损失、边缘拓扑保持损失、梯度方向一致性损失。方法总图可画为：低分辨率MRI -> 浅层特征 -> 主干Transformer分支与高频结构分支并行 -> 频段选择门控融合 -> 解剖先验调制 -> 重建头。消融包括：无解剖先验、无门控、仅频率损失、仅边缘损失、不同高频提取算子、不同噪声水平。
- 预期收益：预期可提升高频重建真实性，减少过锐化和噪声放大，在边界相关指标、视觉评分以及噪声干扰实验中表现更稳。若与原始High-frequency Structure Transformer对比，可突出贡献不在于‘更强网络’，而在于‘更可信的高频建模原则’，有利于形成理论叙事。
- 风险点：若缺少外部解剖先验标注，边缘先验只能从图像自身提取，可能限制增益。；单模态方案虽易验证，但相较多模态任意尺度方案，论文故事的临床应用广度稍弱。；频率损失设计不当可能与像素重建目标冲突，导致PSNR提升不明显。；若数据集本身噪声较低，抑制伪高频的优势不容易显现，需要补充噪声鲁棒实验。
- 证据依据：High-frequency Structure Transformer for Magnetic Resonance Image Super-resolution已证明高频结构先验是有效方向。；其局限性明确指出：高频分支对噪声敏感，缺乏频率-解剖一致性联合约束，可能产生视觉上锐利但医学上不真实的细节。；Dual-domain Multi-modality Brain MRI Arbitrary-scale Super-resolution则反向证明，仅有结构保真主张还不够，需要更明确的约束机制来避免伪结构迁移。；因此，把高频建模从‘增强更多细节’推进到‘增强可信细节’具有直接文献依据。
